# GrahAI Systems — Company Site

Marketing site for **GrahAI Systems**, the parent company behind GrahAI, AasanKhata, and AgencyPitch.

## What's here

```
app/
  layout.js          Root layout — metadata + Organization/WebSite schema + Inter font
  page.js            Homepage — hero, products grid, values, stats, contact, footer
  globals.css        Tailwind layer + brand utilities (.glass, .text-brand-gradient, .bg-grid)
  sitemap.js         Single-route sitemap
  robots.js          Allowlist for AI search bots (GPTBot, ClaudeBot, PerplexityBot, etc.)
public/
  logo.png           1024×1024 brand mark (generated by gemini-3-pro-image-preview)
  logo-square.png    OG / favicon alias
  llms.txt           Canonical company pitch for AI engines
  llms-full.txt      Deep company brief for AI deep-research crawls
scripts/
  generate-logo.mjs  Re-runnable logo generator using the Gemini API
```

## Run locally

```bash
npm install
npm run dev
# Visit http://localhost:3000
```

## Build

```bash
npm run build
npm start
```

## Regenerate logo

```bash
GEMINI_API_KEY=... npm run generate-logo
```

## Stack

- **Next.js 15** (App Router) + **React 19**
- **Tailwind CSS 3.4** — dark theme, glass cards, gradient accents
- **Inter** (Google Fonts, swap display strategy)
- **lucide-react** for icons

## Brand assets

Logo generated via Gemini 3 Pro Image Preview on 2026-05-19.  
Prompt is in `scripts/generate-logo.mjs` — re-run to regenerate variants.

The logo is a stylized "G" with an orbital ring motif, evoking "Grah" (Sanskrit: planet/celestial body) — the etymological root of GrahAI and a callout to the astrology origins of the company's first product.
